Encryption and Decryption - RSA, AES, and Base64 encode and decode using JavaScript

Overview

Welcome to the CS682-Project4b code document! This document describes the code related information about the Project 4b of the CS682 course. In this project, we have implemented module for encryption and decryption using RSA, AES, and Base64 encoding/decoding using JavaScript. The project can be used as a library in the javascript sdk by the Atsign company.

Please refer to the side nav bar for the documentation, which describes the full range of attributes and functions.

About The Project:

  1. Language Used : Javascript
  2. Built and Tested Language: NodeJS.
  3. Version of NodeJS: v18.14.0
  4. Testing Framework Used: Mocha and Chai JavaScript test framework

Available Features

  1. Base64 encode and decode data.
  2. Generate RSA key pairs for encryption and decryption.
  3. Encrypt and decrypt data using RSA encryption.
  4. Signing and Verification process in RSA.
  5. Generate AES keys for encryption and decryption.
  6. Encrypt and decrypt data using AES encryption.
